    CSS

    1. 作用

    CSS是层叠样式表的缩写，作用主要是设置HTML页面中的文本内容（字体、大小、对其方式等），图片的外形（宽高、边框样式、边距等），以及版面布局和外观显示样式。

    2. 一个例子

    <style>
    h2 {
        color: red;
        font-size: 12px;
    }
</style>


<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>

    3. 语法规范

    CSS写在 <head> 标签中间，使用 <style> 标签来包裹，它的语法主要有两个部分构成：选择器、一条或多条的声明

    比如前面那个例子中，h2就是一个选择器，后面的color和font-size就是声明。

    • 选择器：就是需要指定样式的HTML的标签
    • 声明：具体的样式，每一条属性之间都需要加一个分号来分割

    选择器

    由于CSS是针对某个标签进行样式的修改，因此需要首先通过选择器将这些标签给选出来，但是有时候并不希望将某个标签的所有内容都改成同一个样式，这时候需要考虑不同的选择方式。

    选择器又分为基础选择器和符合选择器两种

    1. 基础选择器

    基础选择器是由单个选择器组成的，其中包括四种类型，其中类选择器使用的最多

    下面是黑马的总结

    选择器名 作用 特点 使用场景 用法
    标签选择器 选出所有相同的标签，比如p 不能差异化选择 较多 p { color: red; }
    类选择器 可以选出一个或多个标签 可以根据需求选择 最多 .nav { color: red; }
    id选择器 一次只能选一个标签 只能选择一个 一般需要与js搭配 #nav { color: red; }
    通配符选择器 选择页面中所有的标签 将所有标签全选 特殊情况使用 * { color: red; }

    1.1 标签选择器

    将HTML标签名作为选择器，为页面中所有的这一个标签指定统一的CSS样式。

    缺点也是它会选择所有的标签。

    标签名{
    属性1: 值1;
    属性2: 值2;
}

    例如下面对p标签进行选择指定样式：

    p {
    color: red;
}

    1.2 类选择器（最常用）

    可以单独地选择一个或几个标签来进行样式设计，需要注意的是类名不能是标签的名字。

    .类名{
    属性1: 值1;
    属性2: 值2;
}

    例如下面，将拥有red类的HTML元素都设置为红色

    .bianse{
    color: red;
}

    这时候需要在html中的标签中，设置 class 属性即可

    <div class="bianse"> 红色 </div>

    下面是一个例子：

    在style中设置两个类，分别为red和green，此时在body中加入div，这些div的class就使用这两个类，即可得到下面的效果

    <head>
     <!-- 此处省略前面内容 -->   
    <style>
        .red {
            height: 20px;
            width: 100px;
            background-color: red;
        }

        .green {
            height: 20px;
            width: 100px;
            background-color: green;
        }
    </style>
</head>

<body>
    <div class="red"></div>
    <div class="green"></div>
    <div class="red"></div>
</body>

    多类名：可以给一个标签指定多个类名，从而达到更多的选择目的，这些类名都可以选出这个标签。

    这个多类名的意义也就是解耦合，比如有些类是专门用来控制颜色的，有些类是专门用来控制大小的，将功能不同的类写到不同的类中，这样就实现了解耦合的作用，否则全部写到一起，高度耦合会导致复用性很差。

    下面就是个例子，还是上面的那个效果图，只不过是将大小与颜色的设置类给分开

    <head>
    <style>
        .red {
            background-color: red;
        }

        .green {
            background-color: green;
        }

        .size20-100 {
            height: 20px;
            width: 100px;
        }
    </style>
</head>

<body>

    <div class="red size20-100"></div>
    <div class="green size20-100"></div>
    <div class="red size20-100"></div>
</body>

    1.3 id选择器

    这里是使用id来指定样式，在CSS中，id选择器使用 # 来定义。注意，与类选择器不同的是，id相当于标签的身份证号，每个标签只能有一个id，因此每个设置的id样式只能对于一个id的标签生效，对于其他标签是不可用的。

    id选择器常用语页面上唯一性的元素上，经常与JS一起配合使用。

    #id名{
    属性1: 值1;
    属性2: 值2;
}

    例如下面

    #nav {
    color: red;
}

    这时候，同样需要在HTML中需要指定样式的标签里加上id，并指定id的名字。

    <div id="nav">test</div>

    1.4 通配符选择器

    通配符选择器使用 * 来定义，表示选取页面中的所有标签。也不用调用了，设置完直接就可以用了。

    * {
    属性1: 值1;
    属性2: 值2;
}

    由于威力太强大，一般是在特殊情况下才会使用的一种选择器，比如下面的这个，清除所有元素标签的内外边距

    * {
    margin: 0;
    padding: 0;
}

    2. 复合选择器

    复合选择器就是由多个基本选择器组合而成的。

    2.1 后代选择器

    假设在body中有ol和ul，在ol和ul中都包含了若干个li，此时如果只想要对ol中的li进行选择，此时可以使用后代选择器。

    使用的方式就是用空格将父子关系的元素隔开即可。注意：

    • 父子关系可以叠加多代，也就是说在子代中还包含了其他标签，也会设置为相同的样式。
    • 可以使用类选择器通过类名来写

    <ol class="nav">
    <li>ol的li</li>
    <li>ol的li</li>
    <li>ol的li</li>
</ol>
<ul>
    <li>ul的li</li>
    <li>ul的li</li>
    <li>ul的li</li>
</ul>

    .nav li {
    color: red;
}

    2.2 子选择器

    也叫作子元素选择器，只能选择作为某元素的最近一级子元素，与孙子没关系。

    与后代选择器不同的是，两个元素之间是使用 > 进行连接的。

    <div class="nav">
    <a href="#">儿子</a>
    <p>
        <a href="#">孙子</a>
    </p>
</div>

    .nav>a {
    color: red;
}

    2.3 并集选择器

    可以选择多组标签，同时为他们定义相同的样式。也就是用于集体声明。

    并集选择器通过逗号连接而成，任何形式的选择器（包括后代选择器）都可以作为冰机选择器的一部分

    下面这个div p就是后代选择器，一般来说是将不同的标签分行显示

    ul, 
div p {
    样式设置
}

    2.4 伪类选择器

    用于向某些选择器添加特殊的效果。

    伪类选择器的语法是使用冒号表示，比如:hover、:first-child等。

    伪类选择器有两种，一种是链接伪类选择器，一种是focus伪类选择器。

    1. 链接伪类选择器

      在网页中，对于某一个超链接，在不同的状态下需要有不同的展示效果。

      • link：选择所有未被访问的链接
      • visited：选择所有已被访问的链接
      • hover：选择鼠标指针位于其上的链接
      • active：选择活动链接（鼠标按下未弹起）

      例如下面对链接标签a进行设置，对其各种不同状态的颜色都进行设置。（当然其他的也可以设置，比如下划线等）

      a:link {
    color: #666;
}
a:visited {
    color: #FF0
}
a:hover {
    color: #F0F
}
a:active {
    color: #0F0
}

      注意事项：

      1. 为了保证能够生效，需要按照lvha的顺序进行声明（就是上面的顺序）。

      2. 因为a在浏览器中用于表示链接，它具有默认的样式，因此一般都需要给这个链接单独指定样式。

      3. 在项目中常用的写法是，先指定a，然后指定hover，这样的效果就是鼠标经过时为蓝色，其他情况下是黑色。

        a {
    color: gray;
    text-decoration: none;
}

a:hover {
    color: blue;
}

    1. focus伪类选择器

      这个伪类选择器是用来获取焦点的表单元素，因此主要针对于表单使用该选择器。

      例如下面这个例子，没有选中（获得焦点，得到光标）这个输入框的时候，颜色是白色，选中之后就变成黄色了。

      input:focus {
    background-color: yellow;
}

    字体属性

    用于设置字体、大小、粗细、样式（比如倾斜等）

    1. 设置字体

      设置字体使用 font-family：

      • 对于字体的设置，可以设置多个字体，只需要在字体之间加上逗号即可，多个字体按照先后顺序查找，直到找到安装了的字体。
      • 如果字体的名称中有空格，需要加上引号包起来。
      • 通常对body标签进行设置，因为浏览器内的内容都是放在body里面的，可以起到全局设置的作用。
      body {
    font-family: "Times New Roman", "Microsoft YaHei";
}

    1. 设置字体大小

      使用 font-size 来设置字体大小，谷歌浏览器中默认的文字大小为16px。标题标签（<h>）是比较特殊的，需要单独进行设置大小。

      p {
    font-size: 20px;
}

    1. 设置字体粗细

      使用 font-weight 设置字体粗细，其中包含了集中不同的属性，可以设置为：

      • normal：相当于数字值设置为400
      • bold：相当于数字值设置为700
      • bolder
      • lighter
      • 数字值：100-900之间的整百数值，并且这个数字后面不需要加单位。
      p {
    font-weight: 400;
}

    1. 设置字体样式

      使用 font-style 设置字体样式，主要是文字的风格，属性值包括：

      • normal：正常显示文字
      • italic：显示斜体问题
      p {
    font-style: italic;
}

    1. 设置复合属性

      如果不想在设置的时候将其写成4行的形式，则可以通过复合的方式进行设置，需要注意的是，设置复合属性时，需要遵循固定的顺序，且用空格隔开。

      如果不需要设置其中的某些属性，可以省略，但是 font-size 和 font-family 这两个属性必须要设置，否则不生效。

      具体的顺序如下：

      font: font-style font-weight font-size/line-height font-family;

      div {
    font: italic 700 16px 'Microsoft YaHei';
}

    文本属性

    1. 文本颜色

    使用 color 属性进行颜色设置。颜色有三种方式可以设置：

    • 使用预设的颜色
    • 使用16进制的形式
    • 使用rgb颜色设置，其中前三个是RGB属性，第四个属性为颜色的透明度，最大为1.
    div {
    color: red;
    color: #FF0000;
    color: rgb(255, 0, 0, 0.99);
}

    2. 文本对齐

    使用 text-align 属性来设置元素内文本的水平对齐方式，可以设置为以下这些值：

    • center：居中
    • left：左对齐（默认）
    • right：右对齐
    div {
    text-align: center;
}

    3. 文本装饰

    使用 text-decoration 属性设置文本的修饰，比如给文本添加下划线、删除线、上划线等，可以设置为：

    • none：默认
    • underline：下划线
    • overline：上划线
    • line-through：删除线
    p {
    text-decoration: underline;
}

    4. 文本缩进

    使用 text-indent 属性来制定文本第一行的缩进，通常就是段落首行的缩进

    可以设置两种不同的值，一种是像素大小，一种是相对大小（根据当前文字的大小）

    p {
    text-indent: 10px;
    text-indent: 2em;
}

    5. 行间距

    使用 line-height 属性来设置行间距。行间距包含了文字高度、上下间距，行间距则是这三部分的和，如果改变行间距，不会改变文字的高度，智慧通过改变上下间距的方式进行修改。

    p {
    line-height: 26px;
}

    引入方式

    按照引入方式的不同，可以将CSS的代码分为三种：

    内部样式表、外部样式表、行内样式表。

    1. 内部样式表（嵌入式引入）

      就是写在HTML页面内部，将所有点的CSS代码抽取出来，一般来说都是单独放在 <style> 标签之中的。

    2. 外部样式表

      将样式单独写在css文件中，然后将css文件引入到html页面中来使用。

      在css文件中不需要style标签，直接写样式就行了。

      引入的方式是在html中使用 <link> 标签引入，其中的rel表示样式表。

      <link rel="stylesheet" href="css文件路径">

    3. 行内样式表

      在标签内部通过类似于 style="color='red';" 这种形式来引入，可以控制当前标签的样式。
