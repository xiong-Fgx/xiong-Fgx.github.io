    Emmet语法

    Emmet语法使用缩写来提高HTML和CSS的编写速度。

    1. 快速生成HTML结构语法

    • 生成标签时直接输入标签名，然后按tab即可补全标签
    • 需要生成多个标签时，加上 * 即可
    • 如果有父子关系时，使用 >，比如 ul > li
    • 如果有兄弟关系时，使用 +，比如 div + p
    • 生成带有类名或者id名字的，直接写 .demo 或者 #demo 即可
    • 如果生成的div类名有顺序，可以用自增符号 $，比如 .demo$*5
    • 如果想要在生成的标签内部写内容，则使用 {}，并且把内容写进去

    2. 快速生成CSS样式语法

    CSS也可以简写，就是在写每个属性的时候，只写单词首字母即可

    • 比如 line-height: 26px; 可以直接用 lh26 进行补全
    • 比如 text-decoration: none; 可以直接使用 tdn 进行补全

    元素显示模式

    元素的显示模式指的是元素（标签）以什么方式进行显示，比如 <div> 自己占一行，多个 <span> 可以放在同一行。

    HTML的元素主要分为两种：

    1. 块元素

    块元素：一行只能放一个块元素

    常见的块元素有：

    • <h1> ~ <h6>：文字类块元素
    • <p>：文字类块元素
    • <div>：
    • <ul>：
    • <ol>：
    • <li>：

    特点：

    1. 自己独占一行
    2. 高度宽度、外边距内边距均可控制
    3. 宽度默认是其容器的100%
    4. 是一个容器及盒子，里面可以放行内或者块级元素。此外，文字类的块元素内不可以放其他的块级元素（如h与p），特别是不能放div。

    2. 行内元素

    行内元素：一行可以放多个，有些地方行内元素也叫作内联元素。

    常见的行内元素有：

    • <a>：链接，需要注意的是，链接里面不能再放链接了，但是可以放块级元素，不过将a转换成块级模式是最安全的。
    • <strong>：
    • <b>：
    • <em>：
    • <i>：
    • <del>：
    • <s>：
    • <ins>：
    • <u>：
    • <span>：这个是最典型的行内元素。

    特点：

    1. 相邻的行内元素在同一行上，一行可以显示多个
    2. 高度、宽度的直接设置是无效的
    3. 默认的宽度就是它内容本身的宽度
    4. 行内元素只能容纳文本或者其他的行内元素。

    3. 行内块元素

    行内块元素：

    在行内元素中，有几个特殊的标签，他们同时具有块元素与行内元素的特点，比如：

    • <img/>：图像
    • <input/>：表单
    • <td>：

    特点：

    1. 与相邻的行内元素（行内块）在同一行，但是他们之间会有空白缝隙，一行可以显示多个
    2. 默认宽度就是它本身内容的宽度
    3. 高度、行高、外边距和内边距都可以控制（块级元素特点）

    4. 总结

    元素模式 排列 设置样式 默认宽度 可包含内容
    块级元素 一行只能由一个块级元素 可以设置宽度和高度 其容器的100% 容器级可以包含任何标签
    行内元素 一行可以放多个行内元素 不可以直接设置宽度和高度 其本身内容的宽度 容纳文本或者其他行内元素
    行内块元素 一行可以放多个行内块元素 可以设置宽度和高度 其本身内容的宽度

    5. 显示模式转换

    特殊情况下，我们需要元素模式的转换，简单理解为：一个模式的元素需要另一种模式的特性。

    比如想要增加链接 <a> 的触发范围。

    例如：

    • display:block：转化为块级元素

      例如下面，链接标签a本来是属于行内元素的，不能够定义宽度高度，如果想要这么设置，就需要将一个链接a转化成块级元素。

      设置成块级元素之后，就可以设置宽度和高度，并且独占一行。

      a {
    width: 150px;
    height: 50px;
    blackground-color: red;
    display: block;
}

    • display: inline：转化为行内元素

      例如下面，div本来是块级元素，可以定义宽高，并且独占一行，但是可以通过设置，将其转换为行内元素。

      设置成行内元素之后，宽高设置就无效了，并且可以在一行内显示多个。

      div {
    /*设置之后下面的宽高设置其实已经失效了，不写也一样*/
    width: 150px;
    height: 50px;
    blackground-color: red;
    display: inline;
}

    • <display: inline-block>：转换为行内块元素

      例如下面，span是典型的行内元素，可以通过设置，将其转换为行内块元素。

      设置之后，它就具有了行内块元素的特点，可以同时设置宽高，并且在一行内可以显示多个

      span {
    width: 150px;
    height: 50px;
    blackground-color: red;
    display: inline-block;
}

    CSS背景

    1. 背景颜色

    background-color 属性定义了元素的背景颜色。

    2. 背景图片

    背景图片的优点是很大，方便控制位置。

    设置背景图片的时候，需要使用 background-image 属性来设置，其属性需要加上 url() 才能生效。

    使用 background-repeat 属性对图片的平铺模式进行设置，有一下几种设置的方式：

    • repeat：（默认情况下是平铺的）
    • no-repeat：
    • repeat-x：沿着x轴平铺
    • repeat-y：沿着y轴平铺

    注意：页面元素既可以添加背景颜色，也可以加入背景图片，但是背景色是在下面，背景图片在上面。

    div {
    background-image: url(图片路径);
    background-repeat: no-repeat;
}

    3. 背景图片位置

    使用 background-position 属性来设置背景图片的位置，设置的时候，可以使用方位名词，也可以使用精确单位。

    • 精确数值length：可以是百分数，也可以是由浮点数字和单位标识符组成的长度值

      如果只制定了一个数字值，那么一定是x坐标，他的y坐标会居中。

    • 方位名词position：top、center、bottom、left、center、right方位名词。使用方位名词时，两值的前后顺序不限

      如果只指定了一个方位名词，另一个省略，则第二个默认是居中对齐的。

    • 混合单位：混合使用精确数值和方位名词。

    div {
    background-position: x y;
    background-position: top left;
    background-position: 100px 100px;
    background-position: 100px center;
    background-position: center 100px;
}

    4. 背景图像固定（背景附着）

    使用 background-attachment 属性设置背景图像是固定还是随着页面的其余部分滚动。

    可以设置两个属性来制定，后期可以使用该属性做一个视差滚动的效果

    • scroll：滚动
    • fixed：固定
    div {
    background-attachment: scroll;
}

    5. 复合写法

    复合写法是一种简写，没有特定的顺序，一般约定的顺序为：

    background: 背景颜色 背景图片地址 背景平铺 背景图片滚动 背景图片位置

    6. 背景色半透明

    半透明写法使用rgba来设置，其中的第四个属性就是半透明度，从0取到1。

    background: rgba(0,0,0,0.3)

    CSS三大特性

    1. 层叠性

    给相同的选择器设置相同的样式，此时一个样式就会覆盖另一个冲突的样式，主要用于解决样式冲突的问题。

    原则：

    • 样式冲突，采取的原则是就近原则，哪个样式离结构近，就执行哪个样式。（就近原则）
    • 样式不冲突，不会层叠。

    例如下面这段代码，两个div样式中的color属性冲突，但是font-size并不冲突，因此color会层叠，但是font-size不会层叠。

    div {
    color: red;
    font-size: 12px;
}

div {
    color: blue;
}

    2. 继承性

    子元素可以继承父元素的样式，用来简化代码，降低CSS格式复杂性。但是继承的内容也是有一定限制的，只有以下的这些会被继承：

    • text- 开头的
    • font- 开头的
    • line- 开头的
    • color

    3. 优先级

    当一个元素指定多个选择器时，就会有优先级的概念

    • 选择器相同，则执行层叠性。
    • 选择器不同，则根据选择器的权重执行。

    选择器的权重如下所示：

    选择器 权重
    继承或* (0,0,0,0)
    元素选择器 (0,0,0,1)
    类选择器，伪类选择器 (0,0,1,0)
    ID选择器 (0,1,0,0)
    行内样式 style="" (1,0,0,0)
    !important 重要的（在样式后面直接加!important） 无穷大

    注意：

    • 继承的优先级是0。虽然父元素的选择器权重可能很高，但是子元素继承过来的都是0

    • 对于a链接，浏览器默认制定了一个样式，是蓝色有下划线，因此继承是无效的，比如把a链接放进body，并指定body的样式，此时a将不会从body中继承样式。

      复合选择器中有权重叠加的问题。

    ul li {
    color: green;
}

li {
    color: green;
}

    此时的 ul li 的权重为 0,0,0,1 + 0,0,0,1 = 0,0,0,2 （注意，权重叠加不会进位），因此它的优先级是高于 li 的。
