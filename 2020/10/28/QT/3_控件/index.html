    1. 按钮

    1.1 Push Button

    按钮

    1.2 Tool Button

    工具按钮

    QToolButton类提供了用于命令或选项可以快速访问的按钮，通常可以用在QToolBar里面。

    工具按钮和普通的命令按钮不同，通常不显示文本，而显示图标。

    1.3 Radio Button

    单选按钮，默认所有的radio button互斥，如果想要让这些button按照组进行互斥，则需要通过buttonGroup操作

    QButtonGroup *bg = new QButtonGroup;
bg->addButton(ui->radioButton);
bg->addButton(ui->radioButton_2);

QButtonGroup *bg1 = new QButtonGroup;
bg1->addButton(ui->radioButton_3);
bg1->addButton(ui->radioButton_4);

    1.6 Dialog Button Box

    对话框按钮box，如果只是普通的OK和Cancel，则可以这样写：

    buttonBox1 = new QDialogButtonBox(QDialogButtonBox::Ok | QDialogButtonBox::Cancel, this);
connect(buttonBox1, SIGNAL(accepted()), this, SLOT(button1()));
connect(buttonBox1, SIGNAL(rejected()), this, SLOT(button2()));

    由于Dialog Button中还有很多其他的按钮形式，如果想要使用这些，则应该这样写：

    （下面的例子是用Help按钮）

    buttonBox1 = new QDialogButtonBox(QDialogButtonBox::Ok | QDialogButtonBox::Help, this);
connect(buttonBox1, SIGNAL(accepted()), this, SLOT(button1()));
connect(buttonBox1->button(QDialogButtonBox::Help), SIGNAL(clicked()), this, SLOT(button2()));
